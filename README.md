# Sistema da Empresa Fictícia de Desenvolvedoras Mulheres

Este projeto representa um sistema desenvolvido para uma **empresa fictícia composta por desenvolvedoras mulheres**.  

O objetivo do projeto é **dar mais destaque às mulheres na área de tecnologia**, promovendo visibilidade e representatividade no mercado de trabalho.  

## Tecnologias
- HTML  
> Este foi meu **primeiro projeto e experiência utilizando HTML**.

## Contribuição Individual
Minha participação no projeto incluiu:
- Desenvolvimento da seção de **Contato**
- Implementação do sistema de **Feedback**
- Criação do **Portfólio Pessoal**  

## Observações
Este projeto foi desenvolvido como parte de um **projeto acadêmico em grupo**.
